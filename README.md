# Back2Basics-UIKit

WIP Basic project to test good coding practices in iOS using UIKit. The project designed to establish a basline of features to test architecture or coding practices.

## Architecture

* MVVM
* Coordinators

## UI

* Autolayout
* Interface Builder
* Nib/Xib
* RxCocoa

## Libraries

* RxSwift
* RxCocoa
* RxTest
* Kingfisher
* SwiftLint
* OCMock

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

* Xcode
* Cocoapods

### Installing

pod install

Open workspace

Run project like normal.

## Running the tests

Standard Xcode tests
